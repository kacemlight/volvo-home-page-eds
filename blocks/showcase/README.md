# Showcase Block

Product showcase with large featured image and content side-by-side.

## Usage

```markdown
# Showcase

![XC90](https://example.com/xc90.jpg)

Volvo XC90 — Luxury SUV

Experience uncompromising safety, elegant design, and advanced technology.

[Explore XC90](https://example.com/xc90)
```

## Content Structure

1. **Row 1:** Product image
2. **Row 2:** Title/heading
3. **Row 3:** Description
4. **Row 4:** CTA button with link

## Layout

- Desktop: Image left, content right
- Mobile: Stacked vertically
