# CTA Block

Call-to-action banner with prominent heading and button.

## Usage

```markdown
# CTA

Ready to Explore Volvo?

[Start Your Journey](https://example.com/explore)
```

## Content Structure

1. **Row 1:** Heading/message
2. **Row 2:** CTA button with link

## Styling

- Full-width blue gradient background
- White button with blue text
- Centered, responsive layout
