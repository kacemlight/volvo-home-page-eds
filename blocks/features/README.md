# Features Block

Multi-column grid of feature cards with icons, headings, and descriptions.

## Usage

```markdown
# Features

🚗 | Performance | Advanced engine technology and aerodynamics
⚡ | Safety | State-of-the-art safety systems
🌱 | Sustainability | Electric and hybrid vehicle options
```

## Content Structure

Each row contains:
1. **Icon/Image** (text or image)
2. **Heading** (h3)
3. **Description** (p)

## Grid Layout

- Desktop: 3 columns
- Tablet: 2 columns
- Mobile: 1 column
