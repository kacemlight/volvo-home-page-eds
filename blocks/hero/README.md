# Hero Block

Full-width hero banner with background image, headline, subheading, and call-to-action button.

## Usage

```markdown
# Hero

![Background Image](https://example.com/hero-bg.jpg)

Your Headline Here

Your subheading or supporting text here.

[Learn More](https://example.com)
```

## Content Structure

1. **Row 1:** Background image URL
2. **Row 2:** Headline (h1)
3. **Row 3:** Subheading (p)
4. **Row 4:** CTA button with link

## Styling

- Full-width container
- Responsive height
- Overlay text on background image
- Mobile-optimized layout
